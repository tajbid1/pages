Which hosting site do I like the most:
I liked using both 000webshost and vdrive. The CSIS server can be a little frustrating to work with as it is unpredictable
when the servers will be down. It is also much more complicated to manage the csis server using putty or the NPPFtp plugin.
000webhost has to be my favorite site as it has a clean user interface and is easy to use.
